# Naloga7 – Spletna aplikacija za upravljanje projektov

## Glavne funkcionalnosti

1. **Prijava in registracija uporabnikov**  
   Uporabniki se lahko registrirajo, prijavijo in odjavijo iz sistema.

2. **Upravljanje projektov**  
   Ustvarjanje, urejanje in brisanje projektov z osnovnimi podatki (ime, opis, rok).

3. **Dodeljevanje nalog članom ekipe**  
   Vodja lahko posamezne naloge dodeli razvijalcem in določi prioriteto.

4. **Sledenje napredka**  
   Prikaz stanja nalog (npr. "To do", "In progress", "Done") na kanban tablo.

5. **Komentiranje in zgodovina sprememb**  
   Uporabniki lahko komentirajo naloge, sistem pa shranjuje spremembe.
